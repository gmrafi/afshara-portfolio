# How to Add Your CV/Resume

## Option 1: PDF File
1. Create a folder named `cv` in your portfolio directory
2. Save your CV as PDF: `Afshara_Tasneem_Zoa_CV.pdf`
3. Place it in the `cv` folder
4. Update the `cvLink` in `index.html`:
   ```javascript
   cvLink: "cv/Afshara_Tasneem_Zoa_CV.pdf"
   ```

## Option 2: Google Drive Link
1. Upload your CV to Google Drive
2. Set sharing permissions to "Anyone with the link can view"
3. Copy the shareable link
4. Update the `cvLink` in `index.html`:
   ```javascript
   cvLink: "https://drive.google.com/file/d/YOUR_FILE_ID/view"
   ```

## CV Content Suggestions

Your CV should include:

### 1. Header
- Full Name
- Contact Information (Email, Phone, LinkedIn, GitHub)
- ORCID ID
- Location

### 2. Education
- BRAC University - B.Sc. in Computer Science & Engineering
- Expected Graduation Year
- GPA/CGPA (if strong)
- Relevant Coursework

### 3. Research Experience
- Conference paper submission details
- Research interests and focus areas
- Collaboration details

### 4. Technical Skills
- Programming Languages: Python, C/C++, HTML/CSS, JavaScript
- Tools & Technologies: Git, GitHub, VS Code
- Areas: Data Analysis, AI/ML basics, Web Development

### 5. Projects
- List your academic and personal projects
- Include brief descriptions and technologies used

### 6. Certifications
- Online courses completed
- Training programs attended

### 7. Achievements
- Academic honors
- Conference submissions
- Notable accomplishments

### 8. Extracurricular Activities
- Clubs and organizations
- Leadership roles
- Volunteer work

### 9. Languages
- Bengali (Native)
- English (Proficient)

### 10. References
- Available upon request (or include if you have academic references)

## CV Tips for CS Students

1. **Keep it concise**: 1-2 pages maximum
2. **Use action verbs**: Developed, Implemented, Analyzed, Collaborated
3. **Quantify achievements**: Where possible, use numbers and metrics
4. **Tailor for audience**: Academic vs Industry focus
5. **Professional format**: Clean, readable fonts and layout
6. **Proofread**: No typos or grammatical errors
7. **Update regularly**: Keep your CV current with latest achievements
